| Operator | Description |Example (int a=5, b=2) | Result |
|----------|-------------|-----------------------|--------|
|+ Addition |a + b |7 |
|- Subtraction |a - b |3 |
|* Multiplication |a * b |10 |
|/ Division |a / b 2 |(integer division) |
|% Modulus |(remainder) a % b |1|
